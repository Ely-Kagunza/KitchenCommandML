# Restaurant ML Service

Machine Learning service for restaurant management system providing predictive analytics.

## Features

- **Demand Forecasting**: Predict order volume by hour/day/week
- **Kitchen Performance**: Estimate prep times and identify bottlenecks
- **Customer Analytics**: Churn prediction and lifetime value estimation
- **Inventory Optimization**: Stock level recommendations and reorder points
